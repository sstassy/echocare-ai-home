
## Project info
AI smart homes that care - 24/7 safety and support for those that need it most

Combines smart home technology with an AI-powered monitoring camera system to create a 24/7 home assistant that recognizes and responds to emergencies in real time. 

Merging health sensors, smart home automation, and health-educated AI, EchoCare aims to make home living safer and more supportive for individuals with high risk of emergencies which require immediate care —while giving families peace of mind.

## What technologies are used for this project?

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
